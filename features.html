Features

  • User Authentication
  • Dynamic Review System
  • User Profile Settings
  • Bookmarking
  • Notification System
  • Search and Content Contribution

User Authentication

Allows users to create accounts, log in, and log out securely.

Dynamic Review System

Displays reviews for various courses and instructors and enables users to write, edit and delete their reviews.

Notice how aveverage rating change as new reviews are added

User Profile Settings

Provides a user profile section where users can view and edit their personal information. Including account termination

Bookmarking

Allows users to bookmark favorite courses or instructors for quick access.

Notification System/ Interacttion system

Allows users to interact with reviews by either liking or disliking & sends in-app notifications to users for various events, such as when someone interacts with their review or when there is a new review on a bookmarked course or instructor.

Search and Content Contribution Functionality

Enables users to search for specific courses or instructors. Also allows users to add new/missing courses or instructors to our database
